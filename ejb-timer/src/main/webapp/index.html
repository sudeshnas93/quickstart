What is ejb-timer quickstart?

The ejb-timer quickstart demonstrates how to use the EJB timer service in WildFly. It creates a timer service that uses the @Schedule and @Timeout annotations.

Access the Application

This example application is not intended to be opened in a browser. Instead, the EJB timer service outputs messages to the WildFly server console. View the EJB timer message in the WildFly server log or Console window.
